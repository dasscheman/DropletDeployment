## Droplet Deployment

`git clone https://github.com/dasscheman/DropletDeployment.git /var/DropletDeployment`
`curl -sL https://github.com/thedevdojo/larasail/archive/master.tar.gz | tar xz && source larasail-master/install`
`larasail su`
`larasail setup`
`exit`
